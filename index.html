Comandos utilizados para os 3 primeiros commits

Comando Função
git init Inicializar o Git no diretório especificado
git add Informar quais arquivos devem ser enviados ao repositório remoto
git commit Prepara os arquivos selecionados com o git add para o envio
definitivo
git remote "Define" o repositório remoto, para onde os arquivos serão enviados
e de onde serão recebidos
git push Envia, de forma definitiva, os arquivos desejados para o repositório
remoto
